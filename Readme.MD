 # ![imagen](https://icons.iconarchive.com/icons/crountch/one-piece-jolly-roger/48/Luffys-flag-2-icon.png) API Piece ![imagen](https://icons.iconarchive.com/icons/crountch/one-piece-jolly-roger/48/Zoro-icon.png)

### Resolvi desenvolver essa API como forma de homenagear todos aqueles que, enquanto esperavam a revelação de (o que é o One Piece), partiram. Caso você conheça alguém que partiu enquanto aguardava o fim do anime, deixe aqui o nome dele(a) como forma de homenagem.

---

### Uso
#### Instalação
```` bash
npm install
````
#### Executando em modo nomal
```` bash
npm start
````
#### Executando em modo dev
```` bash
npm run dev
````
